# 2024-birthday
[2024 신입부원 웹 기초 프로젝트] 권유진

🎂 BirthDay Buddy

생일을 주제로 사용자 간 소통과 참여를 유도하는 웹 커뮤니티 서비스입니다.
사용자는 자신의 생일을 공유하고, 생일 이벤트에 참여하며, 게시글과 댓글을 통해 서로 축하와 소통을 나눌 수 있습니다.

📌 프로젝트 개요

목적:
생일이라는 공감 가능한 주제를 중심으로 커뮤니티 참여를 활성화하고,
사용자 경험을 고려한 인터페이스와 이벤트 시스템을 구현

형태: 웹 기반 커뮤니티 서비스

🎯 주요 기능
1. 생일 테마 UI

생일 분위기를 살린 색상, 레이아웃, 디자인 요소 적용

HTML, CSS를 활용한 직관적이고 반응형 사용자 인터페이스 구성

2. 생일 이벤트 시스템

사용자의 생일을 기반으로 한 이벤트 기획 및 구현

생일 축하 참여, 이벤트 참여 버튼 등 사용자 상호작용 기능 제공

JavaScript를 활용한 동적 이벤트 처리

3. 커뮤니티 기능(UI 구현)

게시글 작성 및 조회 기능

댓글을 통한 사용자 간 소통

커뮤니티 참여를 유도하는 기본적인 게시판 구조 구현

🛠 사용 기술 스택
Frontend

HTML: 페이지 구조 설계

CSS: 생일 테마 UI 디자인 및 스타일링

JavaScript: 사용자 상호작용 및 동적 기능 구현

협업 및 관리

Git / GitHub: 버전 관리 및 협업

💡 기획 의도

누구나 공감할 수 있는 ‘생일’이라는 주제를 통해 커뮤니티 진입 장벽을 낮춤

단순 게시판을 넘어 이벤트 요소를 결합하여 실제 사용자 참여를 유도

프론트엔드 기본기(HTML, CSS, JS)를 활용한 웹 서비스 흐름 이해 및 구현 경험 확보

🚀 향후 개선 방향

사용자 계정 및 로그인 기능 추가

생일 알림 및 자동 축하 메시지 기능

백엔드 연동을 통한 데이터 저장 및 관리

모바일 환경 최적화
